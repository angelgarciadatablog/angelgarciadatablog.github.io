# angelgarciadatablog.com

Sitio web personal de Ángel García. Blog sobre datos, análisis y tutoriales.

**URL:** [angelgarciadatablog.com](https://www.angelgarciadatablog.com)

---

## Stack

- HTML / CSS / JavaScript (vanilla)
- Sitio estático servido con GitHub Pages
- Dominio personalizado vía `CNAME`

---

## Estructura

```
/
├── index.html        # Página principal
├── assets/           # Estilos, scripts y plantillas compartidas
├── img/              # Imágenes
└── <post>/           # Una carpeta por publicación
```

El contenido se genera a partir de plantillas, de modo que todas las publicaciones comparten el mismo diseño.

---

## Desarrollo local

Sin paso de build. Levantar un servidor estático en la raíz:

```bash
python3 -m http.server 8000
```
