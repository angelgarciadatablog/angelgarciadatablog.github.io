Cómo empezar con Claude Code en Mac

datablog claude-code
Actualizado: 2026-06-10

Lo más probable es que ya conozcas a Claude por su chat. Y está bien empezar por ahí. Pero en este post quiero mostrarte que ese chat es apenas la puerta de entrada a algo mucho más grande —y cómo dar tu primer paso hacia ese "algo más grande" desde tu Mac.

1. Claude no es solo un chat

Cuando la mayoría de la gente piensa en Claude, piensa en algo parecido a ChatGPT: abres una página en el navegador, escribes, y te responde. Y hasta cierto punto es exactamente eso. Esa página se llama claude.ai, la abres en Google Chrome (o el navegador que uses) y tienes una conversación con Claude. Punto.

Aquí está lo interesante: claude.ai es solo una de las formas de usar Claude. Es la más conocida y la más simple, pero también la que menos aprovecha lo que Claude realmente puede hacer. Pensar que Claude es "el chat" es como pensar que un smartphone sirve solo para llamar.

2. Claude Code: donde está el verdadero potencial

De todas las formas de usar Claude, hay una donde le sacas el jugo de verdad: Claude Code.

La diferencia es enorme. En el chat, Claude solo puede hablarte. Con Claude Code, Claude puede trabajar contigo: leer tus archivos, escribir y modificar código, ejecutar comandos, organizar carpetas, entender un proyecto completo. Deja de ser alguien a quien le preguntas y se convierte en alguien que hace cosas a tu lado.

Hay un único detalle antes de poder usarlo, y prefiero decírtelo de una vez para que no te tome por sorpresa: Claude Code no está en el plan gratuito.

3. Lo que necesitas para empezar: un plan de pago

Para usar Claude Code necesitas, como mínimo, el plan Pro de Claude —alrededor de $20 al mes.

Sé lo que estás pensando, porque yo pensé lo mismo: veinte dólares al mes suena a bastante, sobre todo si vienes del plan gratuito. Pero pongámoslo en perspectiva: es exactamente el mismo precio que ChatGPT Plus, el plan equivalente de la competencia, que también cuesta unos $20 al mes. No estás pagando un sobreprecio —es lo que cuesta hoy una suscripción seria de IA.

Y aquí está la diferencia que importa: por ese mismo precio, Claude te incluye Claude Code. No estás pagando "un chat un poco mejor" —estás pagando por convertir a Claude en una herramienta que trabaja dentro de tus proyectos reales. Cuando lo ves funcionando, los $20 dejan de parecer caros.

Para conseguir tu plan, entra a tu cuenta de Claude y mejóralo desde la página de planes:

Planes y precios de Claude

Una vez que tengas el plan activo, ya podemos pasar a lo bueno: cómo empezar a usar Claude Code.

4. Claude Code no es una sola cosa: las puertas de entrada

Aquí mucha gente se confunde, así que vamos por partes. "Claude Code" no es un único programa que descargas —es una capacidad de Claude a la que puedes entrar por varias puertas:

  • La terminal (CLI) — Claude Code corriendo directamente en la línea de comandos de tu Mac.
  • La extensión de VS Code — Claude Code dentro de tu editor de código, sin salir de él.
  • La app de escritorio — una aplicación con interfaz gráfica, como cualquier otro programa de tu Mac.

Y, en un terreno aparte, está la Claude API —otro mundo, pensado para construir tus propias aplicaciones sobre Claude. No la tocamos aquí.

Las tres primeras te llevan al mismo Claude Code; cambia la puerta, no el cuarto. Entonces, ¿por cuál empezar?

5. Mi recomendación: empieza por la app de escritorio

Si estás dando tus primeros pasos, te recomiendo la app de escritorio. Es la más intuitiva —se instala y se ve como cualquier aplicación de Mac, sin comandos ni configuraciones raras— y ya trae Claude Code dentro, respetando exactamente las mismas reglas que las otras puertas.

La descargas desde aquí:

claude.com/download

Cuando la abres, te encuentras con tres opciones:

Opción Qué es
Chat El chat de siempre, ahora en una app de escritorio en vez del navegador
Cowork Un modo de trabajo colaborativo con Claude
Code Aquí está Claude Code — el motivo por el que vinimos

Las dos primeras son útiles, pero la que nos importa en esta serie es Code. Ahí es donde Claude pasa de conversar a trabajar.

6. La regla de oro: una sola carpeta

Antes de cerrar quiero dejarte el consejo más importante de todo el post —y el que casi nadie te dice al empezar.

Cada vez que uses Claude Code, conéctalo siempre a la misma carpeta.

Te explico por qué. Vamos a suponer que creas una carpeta llamada segundo-cerebro. Cada vez que abras Claude Code, lo apuntas a esa carpeta. ¿Qué consigues con eso?

  • Todas tus sesiones quedan guardadas ahí.
  • Todos tus proyectos viven ahí.
  • Todo el conocimiento que vas acumulando en cada conversación se queda ahí.

Con el tiempo, esa carpeta se convierte literalmente en un segundo cerebro: un lugar único donde Claude tiene a la mano en qué andabas, qué decidiste y cómo trabajas. Si en cambio abres Claude Code en una carpeta distinta cada vez, ese contexto se dispersa y Claude empieza de cero una y otra vez.

Por ahora quédate con la regla: una carpeta fija, siempre la misma. El porqué profundo —cómo esto convierte a Claude en un agente con memoria— merece su propio post.

7. Lo que viene

Ya tienes lo esencial: entiendes que Claude Code es mucho más que el chat, sabes qué plan necesitas, tienes la app instalada y conoces la regla de la carpeta única.
