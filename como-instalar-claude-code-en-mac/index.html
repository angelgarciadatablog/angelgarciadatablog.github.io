Introducción a Claude Code en Mac

datablog claude-code
Actualizado: 2026-06-09

1. Formas de utilizar Claude

Forma Para qué sirve
claude.ai Chat en el navegador, uso diario, exploración
API Integrar Claude en tus apps y scripts
Claude Code Trabajar con Claude directo en tu terminal y proyectos

2. Qué es Claude Code

Claude Code es el CLI oficial de Anthropic. Te permite trabajar con Claude directamente desde tu terminal, dentro de tus proyectos reales. No es solo un chat — Claude puede leer tus archivos, escribir código, ejecutar comandos y entender el contexto de tu proyecto completo.

Plan Precio Claude Code Para quién
Free Gratis No incluido Explorar claude.ai
Pro ~$20/mes Incluido Uso intensivo, devs
Max ~$100/mes Incluido Uso muy intensivo

Importante: El plan gratuito no incluye acceso a Claude Code. Necesitas mínimo el plan Pro para usarlo.

3. Verificar si ya lo tienes instalado

claude --version
Resultado Qué significa
Muestra una versión (ej: 2.1.81) Ya lo tienes
command not found Hay que instalarlo

4. Instalación

curl -fsSL https://claude.ai/install.sh | bash

Este comando siempre instala la versión más reciente. No hay versión hardcodeada.

Respuesta esperada:

Setting up Claude Code...

✔ Claude Code successfully installed!

  Version: 2.1.91

  Location: ~/.local/bin/claude

  Next: Run claude --help to get started

⚠ Setup notes:
  • Native installation exists but ~/.local/bin is not in your PATH. Run:

  echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc

✅ Installation complete!

Una vez instalado, Claude Code se guarda en:

~/.local/bin/claude

Esto importa porque explica directamente el comando de PATH que viene después. Si el instalador dejó el binario en ~/.local/bin/, entonces esa carpeta es la que hay que agregar al PATH. Si lo hubiera instalado en otro lugar, el comando sería distinto.

Antes de la versión 2.1.15, Claude Code se instalaba con npm y dependía del Node.js del sistema. Desde esa versión el método fue deprecado — el nuevo instalador curl empaqueta su propio runtime, por lo que Claude Code es autónomo y no depende de la versión de Node.js que tengas instalada.

Advertencia post-instalación: PATH no configurado

Es posible que al terminar la instalación veas este mensaje:

~/.local/bin is not in your PATH

Esto significa que aunque Claude Code está instalado, tu terminal aún no sabe dónde encontrarlo. Para resolverlo:

echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc

Este comando resuelve dos cosas distintas:

  1. echo ... >> ~/.zshrc — no modifica el PATH en este momento. Escribe la instrucción en .zshrc, que es el archivo que se ejecuta cada vez que abres una terminal. Así el PATH queda configurado de forma permanente para todas las sesiones futuras.
  2. source ~/.zshrc — aplica ese cambio ahora mismo en la sesión actual, sin cerrar y volver a abrir la terminal.

Si solo corrieras export PATH="$HOME/.local/bin:$PATH" directo en la terminal — sin escribirlo en .zshrc — funcionaría en esa sesión pero se perdería al cerrarla.

Luego verifica que quedó bien:

claude --version

4.1 ¿Claude Code instala Node.js en mi sistema?

No. Antes de la versión 2.1.15, Claude Code se instalaba con npm y dependía del Node.js del sistema. Desde esa versión el método fue deprecado — el nuevo instalador curl empaqueta su propio runtime, por lo que Claude Code es autónomo y no depende de la versión de Node.js que tengas instalada.

Ese runtime no queda disponible para el sistema ni para otras herramientas.

Claude Code → runtime interno propio → solo lo usa él
n8n         → necesita Node.js de nvm → son completamente independientes

Si necesitas Node.js para otras herramientas como n8n, debes instalarlo por separado con nvm. La instalación de Claude Code no te lo proporciona.

4.2 ¿Qué pasa si hay varios usuarios en la misma Mac?

Claude Code se instala en el directorio home de cada usuario. Cada usuario debe instalarlo de forma independiente:

Tu Mac
├── Usuario 1 (angel)
│   └── ~/.local/bin/claude  ✅ tiene Claude Code
│
└── Usuario 2
    └── ~/.local/bin/        ❌ no existe, debe instalarlo por su cuenta

Misma lógica que nvm — cada usuario tiene su propio entorno aislado.

Usuario 1 → ~/.local/share/claude ≈ 544MB
           → ~/.claude            ≈ crece según uso

Usuario 2 → ~/.local/share/claude ≈ 544MB
           → ~/.claude            ≈ crece según uso

Total base ≈ ~1GB solo en binarios

4.3 Desinstalación

# Eliminar el binario
rm -f ~/.local/bin/claude

# Eliminar archivos de versión
rm -rf ~/.local/share/claude

# Eliminar configuración e historial (opcional)
rm -rf ~/.claude
rm ~/.claude.json

El último bloque elimina toda la configuración: herramientas permitidas, servidores MCP e historial de sesiones. Omitirlo si solo se quiere reinstalar.

4.4 Espacio en disco

# Ver cuánto pesa el binario
du -sh ~/.local/bin/claude

# Ver cuánto pesa el runtime
du -sh ~/.local/share/claude

# Ver cuánto pesa la configuración
du -sh ~/.claude
Ruta Peso aprox. Qué contiene
~/.local/bin/claude 0B Symlink (acceso directo al binario real)
~/.local/share/claude ~544MB Binario real + runtime interno
~/.claude ~68MB inicial Configuración, historial, caché

~/.local/bin/claude pesa 0B porque no es el archivo real sino un symlink que apunta a ~/.local/share/claude. El peso real está en los 544MB.

El directorio ~/.claude crece con el uso:

Qué agregas Dónde crece
Skills ~/.claude
Configuración de agentes ~/.claude
Servidores MCP ~/.claude
Historial de sesiones ~/.claude
Actualizaciones de Claude Code ~/.local/share/claude

5. Siguiente paso

Con Claude Code instalado y el PATH configurado, el siguiente paso es ejecutarlo por primera vez. Hay conceptos importantes que entender antes de hacerlo — los cubrimos en el siguiente post.

Gotchas

  • El PATH no se configura solo. Si después de instalar el comando claude no funciona, ejecutar el comando de PATH de la sección 4 y recargar la terminal.
  • No requiere Node.js. Claude Code trae su propio runtime. No instalar Node.js como prerequisito.
  • Un usuario, una instalación. En Mac con varios usuarios, cada uno debe instalar Claude Code en su propio home.
