Ruta del analista de datos

Actualizado: 2026-06-10

Empieza aquí

La base de todo análisis de datos son las tablas. El 70% de lo que hace un analista se reduce a eso: consultar tablas, cruzarlas, filtrarlas y extraer algo útil de ellas.

Pero antes de llegar a la tabla hay preguntas que pocos se hacen desde el principio:

  1. ¿Dónde viven esos datos?
  2. ¿Cómo llego a ellos?
  3. ¿Cómo manejo grandes volúmenes sin que todo colapse?
  4. ¿Qué herramienta uso para mostrar los datos?
  5. ¿Cómo uso la inteligencia artificial para analizar datos?

Esta ruta responde esas preguntas en orden. No es un curso de definiciones — es un camino con entregables. Cada módulo existe porque resuelve un problema real que vas a enfrentar como analista.

Vamos a trabajar con Google Cloud y BigQuery como plataforma central, SQL como lenguaje de consulta, APIs para cargar datos externos y Python para automatizar lo que no tiene sentido hacer a mano.

Antes de arrancar, completa el cuestionario de nivel. Son preguntas cortas — no hay respuestas incorrectas. El objetivo es identificar desde dónde partes para no perder tiempo explicando cosas que ya sabes.
